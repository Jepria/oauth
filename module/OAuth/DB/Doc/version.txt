title: Версии



group: 1.1.0 ( 2020-11-16)

[+] Для обеспечения отказоустойчивости добавлена возможность работы с данными
    пользовательских сессий в резерной БД (в которую реплицируются
    данные из основной БД). При этом загрузка измененных данных по сессиям из
    резерной в основную БД не предусмотрена.



group: 1.0.0 ( 2020-04-03)

[+] Реализовано хранение данных о приложениях, сессиях и ключах для
    авторизации пользователей по стандарту OAuth 2.0.
